Welcome to Project Source, typing in commands gets you through the game. Type in help any time for the commands.

In process, but(HELP):
hack object: hacks the specified object and puts it in your inventory
enter place: enters a specific place.
kill name with object: kill a criminal,civilian or anyone, but spcifiy the object.
inventory: lists everything you are currently carrying
unlock door with object: unlocks door with the object you specify
go direction: attempts to travel in direction (north, east, south, west)
use skill name: makes you use the hidden skills through out the game.
search object: searches the indicated object for items
